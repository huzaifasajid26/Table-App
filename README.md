# Table-App

## About
Table-App is a simple web application that allows users to generate multiplication tables for any number between 1 and 50. The table is displayed dynamically on the webpage.

## Features
- Enter a number from 1 to 50.
- Automatically generates the multiplication table from 1 to 10.
- Styled table items for better readability.

## How to Use
Click on this GitHub link to open the project: [Table-App Repository](https://github.com/huzaifasajid26/Table-App)  
Once the project page opens, you can:
1. Download the repository or open it directly in your browser.
2. Open the `index.html` file.
3. Enter a number when prompted.
4. View the generated multiplication table on the page.

## Files
- `index.html` - Main HTML file.
- `style.css` - CSS file for styling the table and page.
- `script.js` - JavaScript file for generating the table dynamically.
- `Assets/` - Folder containing images or other resources if used.
